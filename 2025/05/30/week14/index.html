本周学习目标

秘塔写作猫

  • 功能定位：文本校对、文章润色、AI辅助创作

  • 目前有四种用法：

    1. 用作一个有AI辅助的在线编辑器

    2. 在效率模式下使用其夸张的文本改写功能（不仅包含续写，缩写，还能改写，实现降重）

    3. 在审阅模式下，用来校对错别字或标点负号，以及新闻、自媒体上审核可能出现的政治性错误（如：行政区划分错误等等）

    4. 以前三种用法为基础，配合AI写作功能来解决头脑闭塞问题（文章续写思路缺失？，论文灵感，工作报告，产品介绍，广告文案）

具体表现：没有思路 不想动笔 有了思路 不想动笔等等

  • 与其他AI体区别 秘塔猫属于主题生成型 （输入标题或主题——生成摘要——列举多条大纲——续写成文）

提示词设计

问题1：刑事犯罪嫌疑人在检察审查起诉、法院审判期间的会见权

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 法规政策，回答：
刑事犯罪嫌疑人在人民检察院审查起诉阶段和人民法院审判阶段，哪些主体有权会见？法律依据是什么？
要求：

  1. 核心答案必须源自 .gov.cn 域名的官方网站发布的 法律、司法解释或部门规章；
  2. 直接引用原文规定，标注信息来源（具体法律名称+条款）；
  3. 提供 全国人大、最高检或最高法官网的直达链接；
  4. 非官方信息（如学术观点）需标注 【非官方说明】；
  5. 注明文件时效性及有效性。

问题2：土地征收公告程序的实施时间及法律依据

提示词：
请根据 中国政府及相关部门官方网站 发布的 土地管理法律法规，回答：
土地征收公告程序最早于哪一年在国家级法律法规中明确要求？具体出自哪部法律/条例的哪一条？
要求：

  1. 核心答案必须源自 自然资源部或国务院官网（.gov.cn）发布的 法律/行政法规原文；
  2. 引用条款内容并标注文件名称+条款号；
  3. 提供 《土地管理法》或国务院条例的官网直达链接；
  4. 非官方信息（如地方实践）需单独标注；
  5. 注明该条款的修订/废止状态（如适用）。

问题3：事故调查组中监察机关、人民检察院的参与条件及法规适用性

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 文件，回答：
《生产安全事故报告和调查处理条例》第二十二条中，监察机关、人民检察院参与事故调查组的具体条件是什么？司法体制改革后检察院职权变化是否影响该条款适用？
要求：

  1. 核心答案必须源自 应急管理部、国务院或最高检官网（.gov.cn）发布的 官方解释/补充规定；
  2. 直接引用参与条件的明文规定，标注文件名称+条款；
  3. 提供 条例原文及配套解释文件的官网直达链接；
  4. 检察院职权调整依据需引用 中央深改委或全国人大文件；
  5. 区分标注官方信息与非官方分析。

问题4：经营性用地与公益性用地的区分标准及类别

提示词：
请根据 中国政府及相关部门官方网站 发布的 土地分类国家标准，回答：
现行法规中经营性用地与公益性用地的明确定义、区分标准及具体类别有哪些？
要求：

  1. 核心答案必须源自 自然资源部官网（.gov.cn）发布的 《土地利用现状分类》（GB/T 21010）或其他国家标准文件；
  2. 直接引用用地类别定义及分类表；
  3. 提供 国家标准全文公开系统或自然资源部文件的直达链接；
  4. 非官方分类（如地方实践）需标注 【非官方说明】。

问题5：违规建筑没收后办理产权登记的补缴费用依据（广东）

提示词：
请根据 广东省政府部门官方网站 发布的 现行有效 法规政策，回答：
《广东省城乡规划条例》第八十四条中，违规加高建筑没收违法收入后办理产权登记，需补缴哪些费用？具体政策依据（文件名称+条款）是什么？
要求：

  1. 核心答案必须源自 广东省住建厅、自然资源厅或省政府官网（.gd.gov.cn）发布的 配套文件/操作细则；
  2. 引用补缴费用的明文规定及计算标准；
  3. 提供 政策原文的官网直达链接；
  4. 注明文件发布日期及有效性。

问题6：违规建筑没收后能否办理房产证及流程（广东）

提示词：
请根据 广东省政府部门官方网站 发布的 现行有效 法规政策，回答：
依据《广东省城乡规划条例》第八十四条，违规加高建筑没收违法收入后，是否可办理房产证？具体办理流程及条件是什么？
要求：

  1. 核心答案必须源自 广东省自然资源厅、住建厅或不动产登记中心官网（.gd.gov.cn）发布的 登记规范/实施细则；
  2. 引用办理条件、流程的明文规定；
  3. 提供 《不动产登记暂行条例》广东实施细则或相关文件的直达链接；
  4. 非官方操作案例需单独标注。

问题7：公路控制区房屋的补偿要求

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 法规政策，回答：
不在征地红线范围内但位于公路建筑控制区内的房屋，是否必须补偿？国家级法律依据（文件名称+条款）是什么？
要求：

  1. 核心答案必须源自 交通运输部或国务院官网（.gov.cn）发布的 《公路法》《土地管理法》及配套法规；
  2. 直接引用补偿范围的明文规定；
  3. 提供 法律原文及司法解释的官网直达链接；
  4. 注明条款时效性及地方执行差异（如有）。

问题8：矿产开采权打包转让的税收政策及上市公司优惠

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 税收法规，回答：
企业将矿产开采权等资产包转让给另一法人公司，转让方需缴纳哪些税种？哪些可免税？上市公司作为受让方有何优惠？请标注文件名称+具体条款。
要求：

  1. 核心答案必须源自 财政部、税务总局官网（.chinatax.gov.cn）发布的 税收法规/优惠政策；
  2. 分税种引用免税/优惠条件（如增值税、契税、企业所得税）；
  3. 提供 《资源税法》《企业所得税法》及配套文件的直达链接；
  4. 上市公司优惠需引用 证监会或财税部门联合文件。

问题9：民办学校超范围办学的处罚规定

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 法规政策，回答：
民办学校超出办学许可证地址范围办学，对相关责任人的处罚措施是什么？具体依据（文件名称+条款）？
要求：

  1. 核心答案必须源自 教育部或省级教育厅官网（.moe.gov.cn/.edu.cn）发布的 《民办教育促进法实施条例》及地方细则；
  2. 直接引用处罚类型（罚款、吊销许可等）及责任人条款；
  3. 提供 法规原文的官网直达链接；
  4. 注明地方差异（如广东需补充《广东省民办学校规范达标计划和品牌提升计划》）。

问题10：上市公司欠税期间的分红限制及法律责任

提示词：
请根据 中国政府及相关部门官方网站 发布的 现行有效 法规政策，回答：
上市公司欠缴税款期间，股东能否分红？若分红是否需追回？是否涉嫌偷逃税罪？请标注法律依据（文件名称+条款）。
要求：

  1. 核心答案必须源自 税务总局、证监会或全国人大官网（.gov.cn）发布的 《税收征管法》《公司法》《刑法》；
  2. 引用分红限制、追回程序及刑事责任认定条款；
  3. 提供 法律原文及司法解释的直达链接；
  4. 区分税务处罚（行政责任）与刑事犯罪界限。
